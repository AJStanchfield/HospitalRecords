# HospitalRecords

Create a Hospital Patient Records System using Binary
Search Trees (BST) to efficiently manage and retrieve patient information within
a healthcare setting. Each patient record is represented as a node in the BST,
facilitating quick and organized access to patient details for medical staff.

Key Components:
## 1. Binary Search Tree Implementation: -AJ

- Develop a Binary Search Tree that would be used for storing the data base of
the patients.
- Each node in the tree should contain information about the patient, which includes
his name, history, date of admission, and his room number.
- Make use of the classes available in the Examples of Chapter 12 (BT and BST) to
develop this project.
