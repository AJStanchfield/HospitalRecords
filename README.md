# HospitalRecords

Project Description: Create a Hospital Patient Records System using Binary
Search Trees (BST) to efficiently manage and retrieve patient information within
a healthcare setting. Each patient record is represented as a node in the BST,
facilitating quick and organized access to patient details for medical staff.
Key Components:
## 1. Binary Search Tree Implementation: -AJ

- Implement a Binary Search Tree data structure to represent the database of
patient records.
- Nodes in the tree should encapsulate patient attributes such as name, medical
history, admission date, and room number.
- Utilize the classes included in the Examples of Chapter 12 (BT and BST) to
complete the implementation of this project.

## 2. Patient Record Management: -AJ

- Develop functions to add new patient records to the BST.
- Implement search functionalities to quickly locate patient records based on
their names, medical histories, admission dates, or room numbers.
- Validate user inputs to ensure accurate book details are provided during book
addition or modification.
- Implement error handling for cases where users attempt to perform operations
on non-existent books.
